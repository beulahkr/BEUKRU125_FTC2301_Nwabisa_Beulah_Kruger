Challenge 7.3

const value = "3"
console.log(value + 4 + value)

The above code does not log 10 because the value "3" is stored as a string
which cannot be used in an arithmetic operation and will instead concantenate
and log 343

Instead the code should look like the following:

const value = 3 //no double quotation marks
console.log(value + 4 + value);
